# moxyacademy-gh-copilot-intro

This repository supports a tutorial video series about GitHub Copilot. 
Links to videomaterial are in Google Classroom in section 'Extras'.

## How to use this repo

- The `exercises/` folder contains the content for each video module.
- Each exercise file includes the relevant walkthrough material and hands-on tasks.
- The `tests/` and `pages/` folders provide the sample project used throughout the exercises.

## Branches with solutions

For some modules, there are separate branches with example solutions which you can checkout to compare your soultions or to get inspiration if you get stuck. The branches are named according to the module number:

- `solution/module-2-logic-cusomtization`
- `solution/module-3-agents-and-skills`
- `solution/module-4-mcp-and-hooks`

Module 1 and 5 do not have solution branches as those were not needed for them.
